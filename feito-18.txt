## OBJETIVO
Otimizar o cards slider para:
1. Cards **menores** (como o basehabitation.com)
2. **Mais cards visíveis** na tela (4 por linha em desktop)
3. **Código mais eficiente** (PHP dinâmico ao invés de 15x HTML hardcoded)

---

## MUDANÇAS REALIZADAS

### 1. ESTRUTURA HTML - front-page.php
✅ Reescrita com PHP dinâmico

Implementação:
```php
<?php
$cards_data = [
  ['title' => 'X', 'subtitle' => 'Y', 'desc' => 'Z'],
  // ... 14 mais
];

foreach ($cards_data as $index => $card):
  // Renderiza cada card dinamicamente
endforeach;
?>
```

Benefícios:
✓ **Redução de 1200+ linhas para ~70 linhas** de HTML
✓ **Dados centralizados** em um array PHP
✓ **Fácil edição** - altere apenas o array
✓ **Sem duplicação** de código HTML
✓ **Manutenção simplificada** - add/remove cards alterando array

Estrutura por card (compacta):
```
.pb-row-cards-slider__slide (swiper-slide)
└── .pb-row-cards-slider__slideBox
    ├── .pb-row-cards-slider__title
    ├── .pb-row-cards-slider__imgBox
    └── .pb-row-cards-slider__textWrap
        ├── .pb-row-cards-slider__subtitle
        └── .pb-row-cards-slider__text
```

### 2. ESTILO CSS - vertz.css
✅ Redesenhado para cards menores

Tamanhos compactos:
- Padding: 16px (mobile) → 20px (desktop)
- Título: 15px → 18px (ao invés de 20-28px)
- Subtitle: 13px → 15px
- Text: 12px → 14px (com text-clamp 2-3 linhas)
- Image: aspect-ratio 4:3

Responsividade otimizada:
- Mobile: 1.2 cards (show partial next)
- Tablet (768px): 2.2 cards
- Desktop (1280px): **4 cards completos**

Gap entre cards: 12px (compacto)

Espaciamento interno:
- Título → imagem: 12px
- Imagem → texto: 12px
- Texto items: 8px (mobile) → 10px (desktop)

Visual:
- Fundo: branco #ffffff
- Sombra: leve 0 1px 4px rgba(0,0,0,0.08)
- Hover: sombra aumenta para 0 4px 12px
- Border-radius: 2px
- Truncamento de texto: 2 linhas (mobile) → 3 linhas (desktop)

### 3. JAVASCRIPT - vertz.js
✅ Função simplificada e otimizada

Configuração Swiper:
- slidesPerView: 1.2 → 2.2 → 4 (responsivo)
- spaceBetween: 12px (constante)
- loop: true (infinito)
- speed: 700ms
- autoplay: 5000ms (5 segundos)

Comportamento:
✓ Mobile: vê 1.2 cards + preview do próximo (incentiva scroll)
✓ Tablet: vê 2.2 cards
✓ Desktop: vê 4 cards completos

- Copiar/colar HTML
