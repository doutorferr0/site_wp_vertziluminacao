## 🔧 COMO ADICIONAR CONTEÚDO DO CLIENTE

### 1. Alterar Logo
```
1. Faça upload da logo em WP Admin → Media
2. Edite header.php, linha X
3. Substitua src placeholder por URL real
```

### 2. Alterar Cores Primárias
```
1. Edite vertz.css, linha ~20
2. Altere --color-primary: #ff3b36 → cor cliente
3. Altere --color-secondary: #efeece → cor cliente
```

### 3. Adicionar Cards Slider (15)
```
1. Edite front-page.php, linhas 197-211
2. Altere array $cards_data
3. Atualize: title, subtitle, desc
4. Adicione imagens em --index style
```

### 4. Preencher FAQ
```
1. Edite front-page.php, seção FAQ
2. Substitua Q&A placeholder por reais
```

### 5. Adicionar Contato
```
1. Instale WPForms ou Formidable Forms
2. Crie form
3. Copie shortcode
4. Paste em front-page.php seção Contact
```
