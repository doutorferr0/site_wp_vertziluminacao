### 3. JAVASCRIPT - vertz.js
✅ Adicionada função initCardsSlider()
   Localização: Após initGallerySwiper() (linhas ~333-385)

   Função implementada:
   function initCardsSlider() {
     - Verifica disponibilidade de Swiper
     - Localiza elementos com classe .pb-row-cards-slider__sliderWrap
     - Cria instância Swiper para cada elemento
     - Configura breakpoints responsivos (768px, 1280px)
     - Ativa autoplay (5s delay, pausável ao clicar)
     - Loop infinito
     - Sincronização de bullets com slides
     - Event listeners para navegação por bullets
   }

   Configuração Swiper:
   - slidesPerView: 1 (mobile) → 2 (tablet) → 3 (desktop)
   - spaceBetween: 0 (mobile) → 12px (tablet) → 20px (desktop)
   - loop: true (volta ao início)
   - speed: 600ms (transição suave)
   - autoplay: 5000ms com disableOnInteraction

✅ Atualizada função init()
   Localização: Linhas ~397-410
   Mudança: Adicionada chamada initCardsSlider() junto com initGallerySwiper()
   - Se Swiper carregou: chama ambas as funções
   - Se Swiper não carregou: aguarda load event e chama ambas
