## DÚVIDAS COMUNS

### "Git não reconhecido"
Instale Git for Windows: https://git-scm.com/download/win

### "Erro de autenticação"
Use Personal Access Token (não senha):
1. GitHub → Settings → Developer settings → Personal access tokens
2. Gere um token
3. Cole no prompt quando pedir senha

### "Arquivo muito grande"
Git bloqueia arquivos >100MB. Remova do .gitignore:
```bash
git rm --cached nome-do-arquivo
git commit -m "Remove large file"
```

### "Quer fazer push + pull depois?"
```bash
git pull origin main
git push origin main
```

## COMANDOS GIT ÚTEIS

```bash
# Ver status
git status

# Ver commits
git log --oneline

# Editar último commit
git commit --amend

# Desfazer último commit (manter arquivos)
git reset --soft HEAD~1

# Ver diferenças
git diff

# Criar novo branch
git checkout -b nome-branch

# Mudar de branch
git checkout main

# Deletar branch local
git branch -D nome-branch

# Fetch + Pull
git pull

# Push com novo branch
git push -u origin nome-branch

# Force push (CUIDADO!)
git push origin --force
```
